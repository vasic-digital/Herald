**Goal.** Wave 6 closed the bidirectional loop (subscriber → CC → reply). Wave 6.5 makes the *content* of that loop honor spec V3 §32.6 (classification) + §32.6 command vocabulary (`Bug:` / `Task:` / `Issue:` / `Query:` / `Question:` / `Request:` / `Help:` / `Status:` / `Continue:` / `Done:` / `Resolve:` / `Reopen:`) — i.e. an inbound text message is deterministically classified into one of the documented types, command-prefixed messages route to fast-path handlers BEFORE CC dispatch, and operator-role commands (`Done:` / `Reopen:`) physically migrate rows between `docs/Issues.md` and `docs/Fixed.md` atomically. Plus outbound attachment support so the bot can send back photos / docs / voice with its replies (the existing `tgram.Adapter.SendReply` signature already accepts an `[]commons.Attachment` slice that Wave 6 deliberately ignored — Wave 6.5 wires it through). After all implementations land, a single **15-scenario comprehensive lifecycle test** drives real Telegram + real CC Opus through every command + every attachment kind + every state transition. Evidence under `docs/qa/HRD-101-lifecycle-<run-id>/`. Tags **v0.5.0**.

**Architecture.** Five production surfaces touched. Two new internal packages.

1. `pherald/internal/inbound/classify.go` (NEW) — deterministic §32.6 classifier. Table-driven prefix/regex matchers in fixed precedence order. No LLM fallback this wave (deterministic-only — Wave 7 can layer LLM). Confidence is always `1.0` for a deterministic match; `0.0` for no match (defaults to `query` per §32.6 Query: row — "natural language ≡ research/info request"). Emits `inbound.Classification{Type, Criticality, Confidence}` consumed by `Dispatcher.Handle` and threaded into `CodeRequest.Classification` so CC sees it in the envelope pre-text (§33.6).

2. `pherald/internal/inbound/issue_opener.go` (NEW) — concrete `IssueOpener` writing to `docs/Issues.md`. HRD-NNN allocator scans both `docs/Issues.md` and `docs/Fixed.md` for the current max number; selects `max+1`. Atomic write via `os.OpenFile(O_CREATE|O_EXCL)` on a sidecar `.lock` file (lockfile guarantees serialisation across concurrent `OpenIssue` calls in the same process — though pherald listen is single-goroutine per channel today, the lockfile is the future-proofing). File mutation pattern: read full `docs/Issues.md` → modify the in-memory representation (prepend the new row under the `## Open` heading) → write to temp file in same directory → `os.Rename(tmp, real)` for atomic-on-POSIX semantics.

3. `pherald/internal/inbound/event_emitter.go` (NEW) — concrete `EventEmitter` that bypasses HTTP loopback. The `Emit(ctx, EventPayload)` method constructs a `commons.CloudEventEnvelope` (per spec §11.0) from the payload + calls `runner.Runner.Run(ctx, ceBytes, claims)` directly — no `net/http` round-trip back through `/v1/events`. Requires Runner dependency injection in `inbound.Config`. Tests use a stub Runner that records the call. If no Runner instance is available at listen-time (operator may run `pherald listen` without `pherald serve` infrastructure attached), `Emit` returns an explicit error — no silent drop.

4. `pherald/internal/inbound/commands.go` (NEW — alongside `classify.go`) — fast-path handlers for `Help:` / `Status:` / `Continue:` / `Done:` / `Resolve:` / `Reopen:`. Triggered in `Dispatcher.Handle` BEFORE CC dispatch when `classification.Type` is one of `help_command` / `status_request` / `continuation_request` / `closure` / `reopen`. The handlers read the relevant local file (`docs/Help.md` if present else a built-in catalogue; `docs/Status.md`; `docs/CONTINUATION.md`) and reply with a summary or the file's leading prose. `Done:` and `Reopen:` perform the atomic two-file migration (Issues ↔ Fixed) and require operator-role authorisation via `HERALD_OPERATOR_IDS` env (comma-separated `channel_user_id` allowlist; a stub for the real V3 §32.10 subscriber-role mapping, marked as TODO HRD-102-W7).

5. `commons_messaging/channels/tgram/send.go::SendReply` (MODIFY) — replace the existing `_ = attachments` no-op with real fan-out. For each `commons.Attachment` in the slice, dispatch by `MIMEType`: `image/*` → `telebot.Photo`; `application/*` (PDF, etc.) → `telebot.Document`; `audio/*` (ogg/opus, mp3) → `telebot.Voice` for `audio/ogg` else `telebot.Audio`; fall-through → `telebot.Document` (the safe everyman). Each attachment is sent as a separate Bot API call; for ≥2 same-typed attachments we COULD use `sendMediaGroup` (cheaper, single Telegram render), but Wave 6.5 keeps it simple — sequential sends, one wire call per attachment. Each send is `bot.Send(chat, telebot.<Type>{File: ...}, opts)` with `opts.ReplyTo` carrying the original message ID (so EVERY attachment is a reply, threaded under the original message). Integration test against `httptest.Server` asserts multipart form bodies.

```go
// Package inbound — classify.go: deterministic §32.6 classifier per Wave 6.5.
//
// §107 anchor: actually inspect the input. Mutation gate M1 plants an
// always-empty return; the table-driven unit test catches it.
package inbound

import (
	"regexp"
	"strings"
)

// classifyRule maps a command prefix (case-insensitive, trim-whitespace) to
// the §32.6 type. Order matters — first match wins. The Criticality is
// extracted separately from explicit keywords (urgent/critical/low priority).
type classifyRule struct {
	Prefixes []string // case-insensitive; matched after trim+lower
	Pattern  *regexp.Regexp // optional; if non-nil, used instead of Prefixes
	Type     string
}

var classifyRules = []classifyRule{
	{Prefixes: []string{"bug:", "issue:"}, Type: "bug"},
	{Prefixes: []string{"task:", "implementation:", "impl:"}, Type: "task"},
	{Prefixes: []string{"investigation:", "investigate:"}, Type: "investigation"},
	{Prefixes: []string{"query:", "question:", "q:", "?"}, Type: "query"},
	{Prefixes: []string{"request:"}, Type: "query"}, // §32.6 row — Request maps to query path
	{Prefixes: []string{"help:", "/help"}, Type: "help_command"},
	{Prefixes: []string{"status:", "/status"}, Type: "status_request"},
	{Prefixes: []string{"continue:", "/continue"}, Type: "continuation_request"},
	{Prefixes: []string{"done:", "resolve:"}, Type: "closure"},
	{Prefixes: []string{"reopen:"}, Type: "reopen"},
	{Prefixes: []string{"override:"}, Type: "override"}, // §32.6 operator type-correction
}

var (
	criticalityCritical = regexp.MustCompile(`(?i)\b(critical|urgent|p0|emergency|sev[-_ ]?1)\b`)
	criticalityHigh     = regexp.MustCompile(`(?i)\b(high|important|p1|sev[-_ ]?2)\b`)
	criticalityLow      = regexp.MustCompile(`(?i)\b(low|trivial|p3|p4|sev[-_ ]?[34])\b`)
)

// Classify inspects text and returns the classification per §32.6.
// Default on no prefix match: Type="query" (per §32.6 "natural language → query").
// Criticality defaults to "middle" if no criticality keyword matches.
// Confidence is 1.0 on prefix match, 0.0 on fall-through (the dispatcher's
// CC call still happens for type=query; the LLM is the second-pass classifier
// even though we keep that LLM-fallback in Wave 7).
func Classify(text string) Classification {
	t := strings.TrimSpace(text)
	lower := strings.ToLower(t)

	c := Classification{Type: "query", Criticality: "middle", Confidence: 0.0}

	for _, r := range classifyRules {
		for _, p := range r.Prefixes {
			if strings.HasPrefix(lower, p) {
				c.Type = r.Type
				c.Confidence = 1.0
```

```go
// DocsIssueOpener writes a new HRD-NNN row to docs/Issues.md and
// allocates the next free HRD number by scanning both Issues.md and
// Fixed.md.
//
// Concurrency: serialised via a sidecar lockfile created with
// os.OpenFile(O_CREATE|O_EXCL). Two concurrent OpenIssue calls in the
// same process will see one acquire the lock + the other spin-wait
// (10ms backoff, 30s ceiling). pherald listen is single-goroutine per
// channel today; the lockfile is the future-proofing for the
// multi-channel-fanout case (V3 §32.2).
type DocsIssueOpener struct {
	IssuesPath string         // canonical: "docs/Issues.md"
	FixedPath  string         // canonical: "docs/Fixed.md"
	LockPath   string         // canonical: "docs/.issues.lock"
	Clock      commons.Clock  // for testable timestamps
}

// OpenIssue allocates the next HRD-NNN, builds the row, prepends it
// under the "## Open" heading of IssuesPath, and writes the file
// atomically via temp+rename.
func (o *DocsIssueOpener) OpenIssue(ctx context.Context, p IssuePayload) error {
	if o.IssuesPath == "" {
		return errors.New("DocsIssueOpener: IssuesPath empty")
	}
	if o.LockPath == "" {
		return errors.New("DocsIssueOpener: LockPath empty")
	}
	if o.Clock == nil {
		o.Clock = commons.SystemClock{}
	}

	if err := o.acquireLock(ctx); err != nil {
		return fmt.Errorf("acquire issues lock: %w", err)
	}
	defer o.releaseLock()

	next, err := o.nextHRDNumber()
	if err != nil {
		return fmt.Errorf("allocate HRD-NNN: %w", err)
	}

	row := o.buildRow(next, p)

	if err := o.prependUnderOpen(row); err != nil {
		return fmt.Errorf("prepend row: %w", err)
	}
	return nil
}

// acquireLock spins up to 30s waiting for the lockfile to be free.
func (o *DocsIssueOpener) acquireLock(ctx context.Context) error {
	deadline := time.Now().Add(30 * time.Second)
	for {
		f, err := os.OpenFile(o.LockPath, os.O_CREATE|os.O_EXCL|os.O_WRONLY, 0o600)
		if err == nil {
			_, _ = f.WriteString(strconv.Itoa(os.Getpid()))
			f.Close()
			return nil
		}
		if !os.IsExist(err) {
			return err
		}
		if time.Now().After(deadline) {
			return errors.New("DocsIssueOpener: lock held >30s — stale?")
		}
		select {
		case <-ctx.Done():
			return ctx.Err()
		case <-time.After(10 * time.Millisecond):
		}
	}
}

func (o *DocsIssueOpener) releaseLock() {
	_ = os.Remove(o.LockPath)
}

var hrdNumberRE = regexp.MustCompile(`HRD-(\d+)`)

// nextHRDNumber scans both IssuesPath and FixedPath for HRD-NNN tokens
// and returns max+1.
func (o *DocsIssueOpener) nextHRDNumber() (int, error) {
	max := 0
	for _, p := range []string{o.IssuesPath, o.FixedPath} {
		if p == "" {
			continue
		}
		data, err := os.ReadFile(p)
		if err != nil {
			if os.IsNotExist(err) {
				continue
			}
			return 0, err
		}
		for _, m := range hrdNumberRE.FindAllStringSubmatch(string(data), -1) {
			n, _ := strconv.Atoi(m[1])
			if n > max {
				max = n
			}
		}
	}
	return max + 1, nil
}

// buildRow constructs a single Markdown table row matching the
// docs/Issues.md "Open" table schema.
func (o *DocsIssueOpener) buildRow(n int, p IssuePayload) string {
	today := o.Clock.Now().UTC().Format("2006-01-02")
	title := strings.ReplaceAll(p.Title, "|", `\|`)
	return fmt.Sprintf("| HRD-%03d | %s | open | %s | %s | %s | %s | Catalogue-Check: TBD |\n",
		n, p.Type, p.Criticality, title, today, today)
}

// prependUnderOpen inserts row immediately after the "## Open" heading
// and the table-header / table-separator lines. Writes via
// temp+rename for atomicity.
func (o *DocsIssueOpener) prependUnderOpen(row string) error {
	data, err := os.ReadFile(o.IssuesPath)
	if err != nil {
		return err
	}
	lines := strings.SplitAfter(string(data), "\n")

	// Find "## Open" + skip the table header + separator (2 lines after).
	insertAt := -1
	for i, ln := range lines {
		if strings.HasPrefix(strings.TrimSpace(ln), "## Open") {
			// Walk forward to the first table data row (skip blank lines, header, separator).
			for j := i + 1; j < len(lines); j++ {
				t := strings.TrimSpace(lines[j])
				if strings.HasPrefix(t, "|---") || strings.HasPrefix(t, "| ID") || t == "" || strings.HasPrefix(t, "Per Universal") {
					continue
				}
				insertAt = j
				break
			}
			break
		}
	}
	if insertAt < 0 {
		return errors.New("docs/Issues.md: '## Open' heading not found")
	}

	var newLines []string
	newLines = append(newLines, lines[:insertAt]...)
	newLines = append(newLines, row)
	newLines = append(newLines, lines[insertAt:]...)

	tmp := o.IssuesPath + ".tmp." + strconv.FormatInt(time.Now().UnixNano(), 10)
	if err := os.WriteFile(tmp, []byte(strings.Join(newLines, "")), 0o644); err != nil {
		return err
	}
	return os.Rename(tmp, o.IssuesPath)
}

func (o *DocsIssueOpener) pathDir() string { return filepath.Dir(o.IssuesPath) }
```

```go
func (c *CommandsConfig) migrateRow(ctx context.Context, srcPath, dstPath, hrdRef, dstStatus string) error {
	srcData, err := os.ReadFile(srcPath)
	if err != nil { return fmt.Errorf("read %s: %w", srcPath, err) }
	dstData, err := os.ReadFile(dstPath)
	if err != nil { return fmt.Errorf("read %s: %w", dstPath, err) }

	row, newSrc, err := removeRow(string(srcData), hrdRef)
	if err != nil { return fmt.Errorf("remove row from %s: %w", srcPath, err) }
	newDst := prependRow(string(dstData), row, dstStatus, c.Clock)

	stamp := nanoStamp()
	srcTmp := srcPath + ".tmp." + stamp
	dstTmp := dstPath + ".tmp." + stamp
	srcBackup := srcPath + ".bak." + stamp

	if err := os.WriteFile(srcTmp, []byte(newSrc), 0o644); err != nil { return err }
	if err := os.WriteFile(dstTmp, []byte(newDst), 0o644); err != nil {
		_ = os.Remove(srcTmp); return err
	}

	rename := c.renameFn()
	if err := os.Rename(srcPath, srcBackup); err != nil {
		_ = os.Remove(srcTmp); _ = os.Remove(dstTmp)
		return fmt.Errorf("backup src: %w", err)
	}
	if err := rename(srcTmp, srcPath); err != nil {
		_ = os.Rename(srcBackup, srcPath); _ = os.Remove(dstTmp)
		return fmt.Errorf("rename src: %w", err)
	}
	if err := rename(dstTmp, dstPath); err != nil {
		// ROLLBACK: restore original src
		_ = os.Remove(srcPath)
		_ = os.Rename(srcBackup, srcPath)
		_ = os.Remove(dstTmp)
		return fmt.Errorf("rename dst (rolled back src): %w", err)
	}
	_ = os.Remove(srcBackup)
	return nil
}

func removeRow(src, hrdRef string) (string, string, error) {
	lines := strings.SplitAfter(src, "\n")
	for i, ln := range lines {
		if strings.HasPrefix(strings.TrimSpace(ln), "| "+hrdRef+" |") {
			row := ln
			out := append([]string{}, lines[:i]...)
			out = append(out, lines[i+1:]...)
			return row, strings.Join(out, ""), nil
		}
	}
	return "", "", fmt.Errorf("row containing %q not found", hrdRef)
}

func prependRow(dst, row, status string, clk commons.Clock) string {
	lines := strings.SplitAfter(dst, "\n")
	insertAt := -1
	for i, ln := range lines {
		if strings.HasPrefix(strings.TrimSpace(ln), "|---") {
			insertAt = i + 1
			break
		}
	}
	if insertAt < 0 {
		return dst + row
	}
	annotated := annotateRowStatus(row, status, clk)
	out := append([]string{}, lines[:insertAt]...)
	out = append(out, annotated)
	out = append(out, lines[insertAt:]...)
	return strings.Join(out, "")
}

var rowStatusRE = regexp.MustCompile(`^(\| [^|]+ \| [^|]+ \|)[^|]+(\|.*\|)[^|]+(\| [^|]+ \|.*)$`)

func annotateRowStatus(row, status string, clk commons.Clock) string {
	if clk == nil { clk = commons.SystemClock{} }
	today := clk.Now().UTC().Format("2006-01-02")
	if m := rowStatusRE.FindStringSubmatch(row); m != nil {
		return m[1] + " " + status + " " + m[2] + " " + today + " " + m[3]
	}
	return row
}

func (c *CommandsConfig) renameFn() func(string, string) error {
	if c.Rename != nil { return c.Rename }
	return os.Rename
}
```

```go
func (a *Adapter) SendReply(ctx context.Context, chatID int64, text string, replyToID int, attachments []commons.Attachment) (int, error) {
	_ = ctx

	if err := a.ensureBot(); err != nil {
		return 0, fmt.Errorf("tgram.SendReply: %w", err)
	}
	if text == "" {
		return 0, fmt.Errorf("tgram.SendReply: empty text")
	}

	chat := &telebot.Chat{ID: chatID}
	textOpts := &telebot.SendOptions{}
	if replyToID > 0 {
		textOpts.ReplyTo = &telebot.Message{ID: replyToID}
	}

	sentText, err := a.bot.Send(chat, text, textOpts)
	if err != nil {
		return 0, fmt.Errorf("tgram.SendReply (text): %w", err)
	}
	if sentText == nil || sentText.ID == 0 {
		return 0, fmt.Errorf("tgram.SendReply: empty Message response (§107 bluff guard)")
	}

	// Attachment fan-out. Each attachment is its own ReplyTo=original
	// call so all sit at the same thread depth as the text reply.
	for i, att := range attachments {
		if att.LocalPath == "" {
			return 0, fmt.Errorf("tgram.SendReply: attachment[%d] LocalPath empty", i)
		}
		media := buildTelebotMedia(att)
		mediaOpts := &telebot.SendOptions{}
		if replyToID > 0 {
			mediaOpts.ReplyTo = &telebot.Message{ID: replyToID}
		}
		if _, err := a.bot.Send(chat, media, mediaOpts); err != nil {
			return 0, fmt.Errorf("tgram.SendReply (attachment[%d] mime=%q): %w", i, att.MIMEType, err)
		}
	}

	return sentText.ID, nil
}

// buildTelebotMedia maps a commons.Attachment to the appropriate
// telebot media struct. Routing by MIMEType prefix:
//
//   image/*       → telebot.Photo
//   audio/ogg     → telebot.Voice (Telegram renders waveform UI)
//   audio/*       → telebot.Audio
//   video/*       → telebot.Video
//   application/* → telebot.Document
//   *             → telebot.Document (safe fallback)
func buildTelebotMedia(att commons.Attachment) telebot.Sendable {
	file := telebot.FromDisk(att.LocalPath)
	caption := att.Filename
	switch {
	case strings.HasPrefix(att.MIMEType, "image/"):
		return &telebot.Photo{File: file, Caption: caption}
	case att.MIMEType == "audio/ogg" || att.MIMEType == "audio/ogg; codecs=opus":
		return &telebot.Voice{File: file, Caption: caption}
	case strings.HasPrefix(att.MIMEType, "audio/"):
		return &telebot.Audio{File: file, Caption: caption, FileName: att.Filename}
	case strings.HasPrefix(att.MIMEType, "video/"):
		return &telebot.Video{File: file, Caption: caption, FileName: att.Filename}
	default:
		return &telebot.Document{File: file, Caption: caption, FileName: att.Filename}
	}
}
```
